##### 力の公式総整理

##### ① 力の基本公式一覧（最重要）

| 分類 | 公式 | 各量の意味 | 単位（SI） | 単位分解 | 性質 |
|:---|:---|:---|:---|:---|:---|
| 運動の基本 | $$\vec{F}=m\vec{a}$$ | 力＝質量×加速度 | N | kg·(m/s²) | ベクトル |
| つり合い | $$\sum \vec{F}=0$$ | 合力が0 | N | kg·m/s² | ベクトル |
| 重力 | $$\vec{F}=m\vec{g}$$ | 地球から受ける力 | N | kg·(m/s²) | ベクトル |
| 電場中 | $$\vec{F}=q\vec{E}$$ | ローレンツ力 | N | C·(N/C) | ベクトル |
| 磁場中 | $$\vec{F}=q\vec{v}\times\vec{B}$$ | ローレンツ力 | N | C·(m/s)·(N·s/C·m) | ベクトル |
| 電流 | $$\vec{F}=BIL$$ | 磁場中の電流 | N | (N/A·m)·A·m | ベクトル |
| 浮力 | $$\vec{F}=\rho g V$$ | 流体から受ける上向きの力 | N | (kg/m³)·(m/s²)·m³ | ベクトル |
| 重量 | $$W=mg$$ | 物体の重さ | N | kg·(m/s²) | 分量 |
| 垂直抗力 | $$N=mg$$ | 水平面 | N | kg·(m/s²) | 分量 |
| 垂直抗力 | $$N=mg\cos\theta$$ | 斜面 | N | kg·(m/s²) | 分量 |
| 張力 | $$T=mg$$ | 静止・等速 | N | kg·(m/s²) | 分量 |
| 摩擦力（最大） | $$F=\mu_s N$$ | 静止摩擦 | N | 無次元×(kg·m/s²) | 分量 |
| 摩擦力（動） | $$F=\mu_k N$$ | 動摩擦 | N | 無次元×(kg·m/s²) | 分量 |
| 弾性力 | $$F=kx$$ | バネの復元力 | N | (N/m)·m | 分量 |
| 万有引力 | $$F=G\frac{m_1m_2}{r^2}$$ | 万有引力 | N | (N·m²/kg²)·kg²/m² | 分量 |
| 向心力 | $$F=m\frac{v^2}{r}$$ | 円運動 | N | kg·(m²/s²)/m | 分量 |
| 向心力 | $$F=mr\omega^2$$ | 角速度表示 | N | kg·m·(1/s²) | 分量 |
| 空気抵抗（低速） | $$F=kv$$ | 速度比例（層流） | N | (N·s/m)·(m/s) | 分量 |
| 空気抵抗（高速） | $$F=kv^2$$ | 速度²比例（乱流） | N | (N·s²/m²)·(m²/s²) | 分量 |
| クーロン力 | $$F=k\frac{q_1q_2}{r^2}$$ | 電気力 | N | (N·m²/C²)·C²/m² | 分量 |

##### ② よく出る「係数・定数」単位表

| 記号 | 意味 | 単位 | 注意・代表値 |
|:---|:---|:---|:---|
| $g$ | 重力加速度 | m/s² | 約 $9.8\ \text{m/s}^2$（入試では $10$ とすることも多い） |
| $\mu_s,\mu_k$ | 摩擦係数 | 無次元 | 物質による（数値は問題文で与えられる） |
| $k$（バネ） | ばね定数 | N/m | バネ固有（数値は問題文で与えられる） |
| $k$（抵抗） | 抵抗係数 | N·s/m | 形状・流体依存（数値は与えられる） |
| $k$（電気） | クーロン定数 | N·m²/C² | $9.0\times10^9$ |
| $G$ | 万有引力定数 | N·m²/kg² | $6.7\times10^{-11}$ |

##### 覚え方（表格活用法）

| 覚え方 | 内容 |
|:---|:---|
| 単位逆算 | N = kg·m/s² に必ず戻す |
| 力は全部N | 名前が違っても単位は同じ |
| 向心力は役名 | 正体は既存の力 |
| 作図→表 | 図を書いて表で確認 |
| 最終確認 | 式 → 単位 → 数値 |

##### 最重要一行
$$
\vec{F}=m\vec{a}
$$
**すべての力学はここに戻る**
