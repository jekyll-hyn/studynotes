【目的】
・入試対策用の教材として使う
・Astro / Markdown で表示する
・「削除線版（修正過程が見える）」の形式で出力する
・日本語で番号付き解説・理由説明を行う
・最後は例文を追加

【出力形式の厳守事項】
1. 添削本文では「原文の修正過程のみ」を示す（評価・講評・感想は書かない）
2. Markdown + HTML の混在を許可する
3. 語句の修正は必ず以下のルールで表記すること
   - 削除される語句：~~<span class="imt1">削除語句</span>~~
   - 修正後の語句：<span class="imt2">修正語句</span>
   - 削除語句の中に <span> を含めてもよい
4. 元の英文の構造・段落・改行は極力維持すること
5. 表現を大きく書き換えず「入試で減点される点のみ」を修正する
6. 修正対象は以下を中心とする
   - 冠詞・前置詞
   - 大文字／小文字
   - 単数・複数
   - 時制
   - 語法・コロケーション
   - 文法的に不自然な表現
7. diff 記法（```diff）は使用しない
8. 完成文のみを出力しない

【CSS 前提】
・.imt1 = 赤（削除語句）
・.imt2 = 青（修正後語句）
